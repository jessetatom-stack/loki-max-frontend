# Loki Max v1.4 Real Data Frontend

Replace your frontend with this package to get:
- real schedule/score data through your backend
- sportsbook-feel cards
- slip system
- leaders/profile/picks views
- tighter, layered UI

Render settings:
- Root Directory: leave blank
- Build Command: npm install && npm run build
- Publish Directory: dist

Environment:
VITE_API_BASE_URL=https://YOUR-BACKEND-URL.onrender.com
